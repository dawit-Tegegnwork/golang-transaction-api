golang-transaction-api

Synthetic wallet REST API — deposits, withdrawals, transfers, idempotency keys.

GET  /health
POST /users
POST /users/{id}/accounts
POST /accounts/{id}/deposit     Idempotency-Key header
POST /accounts/{id}/withdraw
POST /accounts/{id}/transfer
GET  /accounts/{id}/transactions

$ curl -X POST localhost:8080/accounts/ACCT/deposit \
  -H 'Idempotency-Key: demo-1' \
  -d '{"amount":2500,"reference":"payroll"}'
